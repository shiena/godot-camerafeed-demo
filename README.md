# godot-camerafeed-demo

## Requirement

[Godot Engine](https://godotengine.org/)

## Supported platforms

- macOS: Godot Engine 4.0 or later
- Linux: Godot Engine 4.4 or later
- Android: Godot Engine 4.5 or later
